# 🏥 LaboGest

Application professionnelle de gestion de laboratoire médical.

## ✨ Fonctionnalités

- 🔐 Authentification sécurisée
- 👥 Gestion des patients
- 🔬 Catalogue d'analyses
- 📄 Création de devis
- 📊 Tableau de bord
- 🖨️ Génération PDF

## 🛠️ Technologies

- **Backend** : Node.js, Express, MongoDB
- **Frontend** : React, Tailwind CSS, Vite
- **Sécurité** : JWT, Bcrypt

## 🚀 Installation

```bash
# Backend
cd backend
npm install
cp .env.example .env
npm run dev

# Frontend
cd frontend
npm install
npm run dev
